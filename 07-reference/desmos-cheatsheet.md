# Desmos Quick-Reference Card

One page to review before every practice test. Full lesson: [0.6 Desmos Mastery](../00-orientation/06-desmos-mastery.md). Practice in the [testing version](https://www.desmos.com/testing/cb-digital-sat/graphing).

## Instant recipes

| Problem type | Recipe |
|--------------|--------|
| Solve f(x) = g(x) | Graph both sides → click intersections |
| Solve equation = 0 | Type the equation directly (vertical lines at solutions) or graph y = LHS, click x-intercepts |
| System of equations | Type both **exactly as printed** → click intersection |
| "No solution / infinitely many for what k" | Slider on k → watch lines become parallel/identical |
| Tangency ("exactly one solution") | Slider → curve just touches; confirm exact value with discriminant |
| Vertex / max / min | Graph → click the extremum point |
| Evaluate f(a) | Define `f(x) = ...` then type `f(a)` |
| Mean / median / SD | `A = [ ... ]` then `mean(A)`, `median(A)`, `stdev(A)` |
| Line of best fit | Table (+ button) → `y₁ ~ mx₁ + b` |
| Exponential fit | `y₁ ~ a*b^x₁` |
| "Which expression is equivalent" | Graph original AND each choice → identical curves = equivalent |
| Circle from general form | Type `x² + y² + Dx + Ey = F` as-is — Desmos draws it |
| Absolute value | `abs(...)` — count intersections for "how many solutions" |
| Trig values | Check DEGREE mode first (wrench icon)! |

## Speed habits

- Type equations *as printed* — rearranging wastes time and invites sign errors.
- Fractions beat decimals: type `5/12`, answer SPR as `5/12`.
- Zoom out before concluding "no intersection" — solutions hide off-screen.
- Gray points are clickable: intersections, intercepts, extrema.
- `x = 5` draws a vertical line — handy for reading a curve's value at a point.

## When to skip Desmos

- One-step algebra (typing costs more than thinking)
- Pure symbolic manipulation with no numbers
- Final 2 minutes with 3+ questions left — triage first

## The pre-test drill (5 minutes, before every practice test)

1. Solve x² + 2x − 8 = 0 by graph. *(−4, 2)*
2. System 3x − y = 7, x + 2y = 0. *((2, −1))*
3. Slider: k where y = 2x + k is tangent to y = x². *(−1)*
4. `mean([88,92,79,95,91])`. *(89)*
5. Regression through (1,3.1),(2,4.9),(3,7.2),(4,8.8). *(m ≈ 1.96)*
