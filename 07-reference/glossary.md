# Glossary — SAT Math Terms

The SAT is partly a reading test. These are the terms whose precise meaning decides answers.

| Term | Exact meaning | Watch out |
|------|---------------|-----------|
| **Integer** | ..., −2, −1, 0, 1, 2, ... | Includes negatives and 0; excludes fractions |
| **Whole number** | 0, 1, 2, ... | No negatives |
| **Positive / negative** | > 0 / < 0 | **Zero is neither** |
| **Nonnegative** | ≥ 0 | Includes 0 |
| **Consecutive integers** | n, n+1, n+2 | Consecutive *even/odd*: n, n+2, n+4 |
| **Sum / difference / product / quotient** | +, −, ×, ÷ | "Difference of a and b" = a − b (order) |
| **At least / at most** | ≥ / ≤ | Inclusive! "More than" is strict |
| **Solution** | A value making the equation true | "No solution" is a legitimate answer |
| **Zero / root of a function** | x-value where f(x) = 0 | It's an x, not a y |
| **x-intercept / y-intercept** | Point where graph meets that axis (y = 0 / x = 0) | Read which is asked |
| **Vertex** | Turning point of parabola / absolute value graph | "Maximum value" = the y of the vertex |
| **Function is increasing** | y rises as x rises | Can be on an interval only |
| **Undefined** | Denominator = 0 (or even root of a negative) | Where rational functions break |
| **Extraneous solution** | Algebra-created value that fails the original equation | From squaring / clearing denominators |
| **Equivalent expression** | Equal for ALL values of the variable | Desmos: identical graphs |
| **In terms of x** | Answer contains x | "Solve for a in terms of b" — a alone on one side |
| **Constant** | A fixed number (often named k, a, b, c) | Sliders in Desmos |
| **Coefficient** | Number multiplying a variable | In 3x², it's 3 |
| **Directly proportional** | y = kx (both grow together) | Through the origin |
| **Inversely proportional** | xy = k | One grows, other shrinks |
| **Mean / median / mode / range** | Average / middle of sorted / most frequent / max − min | Median needs SORTED data |
| **Standard deviation** | Typical distance from the mean (spread) | Never computed by hand on SAT |
| **Margin of error** | Uncertainty range around a sample estimate | About the population value, not individuals |
| **Random sample** | Every member equally likely to be chosen | Required to generalize |
| **Random assignment** | Subjects randomly placed in treatment groups | Required for causation |
| **Probability** | favorable/total, between 0 and 1 | "Given that" changes the total |
| **Expected number** | probability × group size | A count, not a probability |
| **Congruent** | Same shape AND size | ≅ |
| **Similar** | Same shape, proportional sides, equal angles | Areas scale by k² |
| **Bisect** | Cut exactly in half | Angle or segment |
| **Perpendicular** | Meet at 90° | Slopes multiply to −1 |
| **Tangent (line)** | Touches curve/circle at exactly one point | Discriminant = 0; ⊥ to radius |
| **Inscribed** | Drawn inside, vertices/edges touching | Inscribed angle = ½ central |
| **Arc / sector** | Piece of circumference / pie slice of area | Different formulas |
| **Radian** | Angle unit; π rad = 180° | Desmos mode! |
| **Circumference** | Perimeter of a circle, 2πr | Not the area |
| **Isosceles** | Two equal sides (⇒ two equal base angles) | May have two cases |
