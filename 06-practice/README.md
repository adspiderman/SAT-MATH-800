## Simulation rules (non-negotiable)

1. 35:00 on a timer per module, back to back with a ≤2-minute pause between.
2. [Desmos testing calculator](https://www.desmos.com/testing/cb-digital-sat/graphing) open; paper for scratch work; nothing else.
3. Answer every question. Use Mark-for-Review habits: note question numbers you'd flag.
4. Score after BOTH modules, not between.

## Scoring guide (approximate scaled score)

| Total correct (of 44) | Approx. score |
|-----------------------|---------------|
| 44 | 800 |
| 42–43 | 760–790 |
| 40–41 | 730–760 |
| 37–39 | 700–730 |
| 33–36 | 650–700 |

## The review protocol (as important as the test)

For EVERY question — including correct ones:
1. Was my method the fastest one? (Check the answer explanation's method.)
2. Log every miss in the [error log](../05-strategies/06-error-log.md) with error type and root cause.
3. Re-do all misses cold 3 days later.

Then take an official Bluebook test within the week — official questions are the final calibration ([official resources](../07-reference/official-resources.md)).
