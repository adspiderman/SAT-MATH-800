## The big ideas

1. **Every ratio/rate/percent problem is a proportion** — set up `part/whole = part/whole` with matching labels and units.
2. **Percent change compounds multiplicatively**: +20% then −20% is ×1.2×0.8 = ×0.96, not back to start.
3. **Mean uses totals; median uses position.** Outliers drag the mean, barely touch the median.
4. **In a two-way table, "given that" shrinks your world to one row or column.**
5. **Random sampling → generalize; random assignment → cause.** The two randoms answer different questions.
