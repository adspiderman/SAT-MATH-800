# Lesson 0.3 — Study Plans

Pick the plan matching your timeline. All plans assume the diagnostic is done first and use official Bluebook tests as milestones. **Hours listed are focused, phone-away hours.**

---

## 12-Week Plan (~6–8 hrs/week) — recommended if you have the runway

| Week | Focus | Details |
|------|-------|---------|
| 1 | Orientation | Lessons 0.1–0.6, diagnostic, Desmos drills, set up error log |
| 2 | Algebra I | Lessons 1.1–1.3 |
| 3 | Algebra II | Lessons 1.4–1.5, Unit 1 quiz |
| 4 | Advanced Math I | Lessons 2.1–2.3 |
| 5 | Advanced Math II | Lessons 2.4–2.6 |
| 6 | Advanced Math III | Lessons 2.7–2.10, Unit 2 quiz |
| 7 | Data Analysis | Lessons 3.1–3.7, Unit 3 quiz |
| 8 | Geometry & Trig | Lessons 4.1–4.5, Unit 4 quiz |
| 9 | Strategy | Unit 5 complete; **Bluebook Test 1** (timed); review + error log re-dos |
| 10 | Mixed practice | Course Practice Test 1; error-log re-dos; re-study weakest 2 lessons |
| 11 | Simulation | **Bluebook Test 2**; Course Practice Test 2; Hard Problem Gauntlet |
| 12 | Peak | **Bluebook Test 3**; light review of error log + formula sheet; rest 2 days before test |

| 2 | Finish Algebra (1.4–1.5 + quiz); Advanced Math 2.1–2.2 |

## 4-Week Intensive (~15 hrs/week) — crunch mode

| Week | Focus |
|------|-------|
| 1 | Orientation + diagnostic + Desmos. Full Algebra unit. Start Advanced Math (2.1–2.4). |
| 2 | Finish Advanced Math + quiz. Full Data Analysis unit. **Bluebook Test 1** at week's end. |
| 3 | Full Geometry & Trig unit. All of Unit 5. Course Practice Test 1. **Bluebook Test 2.** |
| 4 | Course Practice Test 2, Hard Problem Gauntlet, **Bluebook Test 3**, error-log re-dos daily. Rest the final day. |

In crunch mode, if the diagnostic shows a strong domain (≥90%), skim that unit's lessons and jump to its quiz.

---

## Weekly rhythm (all plans)

- **Study days (3–5×/week):** one lesson per sitting, full learning loop.
- **Review day (1×/week):** error-log re-dos from ≥3 days ago; skim traps sections of completed lessons.
- **Never end a session on an unresolved miss** — understand it or flag it for tomorrow, don't just move on.

## Milestone score guide

| After | Bluebook Math score you want | If below |
|-------|------------------------------|----------|
| Content phase (Test 1) | 650+ | Re-do weakest unit before continuing |
| Strategy phase (Test 2) | 700–730+ | Focus error log: are misses concept or careless? |
| Final week (Test 3) | 750+ | Drill your top-2 error categories exclusively |

Scores on official Bluebook tests are the truth; course quizzes and tests are training instruments.
