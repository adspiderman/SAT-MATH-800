# Unit 1 — Algebra (~35% of the test, 13–15 questions)

Linear equations, systems, inequalities, and linear functions. This is the largest domain on the SAT alongside Advanced Math, and it contains the highest density of "must not miss" questions: most Algebra questions are easy-to-medium, so at the 750+ level they are all expected points.

## The one big idea

Every linear situation is `output = start + rate × input` (that is, y = b + mx). If you can identify the *start* (y-intercept) and the *rate* (slope) in any story, table, graph, or equation — and translate between all four representations — you own this unit.
