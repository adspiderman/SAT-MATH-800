# SAT-MATH-800 — The Complete Self-Paced SAT Math Course

**Goal: 750+ on the Digital SAT Math section. Target: 800.**

This repository is a complete, self-contained SAT Math course. It covers every skill tested on the Digital SAT Math section, in the same four content domains the College Board uses, with lessons, worked examples, Desmos calculator technique, common traps, unit quizzes, full-length practice modules, and an 800-level "hard problem gauntlet."

## The path (start here)

1. **Read** [00-orientation/01-digital-sat-math-overview.md](00-orientation/01-digital-sat-math-overview.md) — know the test format cold.
2. **Take** the [diagnostic test](00-orientation/04-diagnostic-test.md) to find your weak domains.
3. **Pick** a study plan in [00-orientation/03-study-plans.md](00-orientation/03-study-plans.md) (4-week, 8-week, or 12-week).
4. **Work** through Units 1–4 in order, doing every practice problem *before* looking at solutions. Take each unit quiz.
5. **Learn** the Desmos calculator deeply — [00-orientation/06-desmos-mastery.md](00-orientation/06-desmos-mastery.md). On the Digital SAT this is worth 30–60 points by itself.
6. **Drill** strategy: [05-strategies](05-strategies/), especially the error log system.
7. **Simulate**: take the full practice tests in [06-practice](06-practice/) under real timing (35 min per 22-question module), then do the [hard problem gauntlet](06-practice/hard-problem-gauntlet.md).
8. **Track** everything in [PROGRESS.md](PROGRESS.md).

## Score targets and what they require

| Target | Max questions missed (approx.) | What it takes |
|--------|-------------------------------|---------------|
| 700 | ~6–8 | Solid on Units 1–3; few careless errors |
| 750 | ~3–4 | Strong on all four units; fast Desmos fluency |
| 800 | 0–1 | Everything below + trap awareness + time to double-check |

An 800 is not about knowing exotic math — the SAT tests nothing beyond precalculus basics. It is about **zero careless errors under time pressure**. That is a trainable skill, and this course trains it deliberately (see [05-strategies/06-error-log.md](05-strategies/06-error-log.md)).

## Rules of the course

1. **Do the problem before reading the solution.** Reading math is not doing math.
2. **Every missed problem goes in your error log.** No exceptions. Re-do logged problems 3 days later.
3. **Time yourself** once you finish a unit. Untimed accuracy first, then speed.
4. **Use Desmos the way you will on test day.** Practice in the actual [Desmos testing calculator](https://www.desmos.com/testing/cb-digital-sat/graphing).
5. **Take official College Board practice tests in Bluebook** alongside this course — they are the only perfectly realistic questions in existence. This course teaches you the math and tactics; official tests calibrate you. See [07-reference/official-resources.md](07-reference/official-resources.md).
