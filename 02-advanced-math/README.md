## The big ideas

1. **A quadratic has five faces:** standard form, factored form, vertex form, its graph, and its table. Hard questions are usually just translation between faces.
2. **The discriminant b² − 4ac counts real solutions** (2 / 1 / 0) — and "how many solutions/intersections" questions are everywhere on hard Module 2.
3. **Exponential = constant ratio; linear = constant difference.** Spot which one a table or story describes.
4. **Extraneous solutions** appear whenever you square an equation or clear a denominator. Always check candidates against the original.
5. **Desmos verifies all algebra**: graph the original and your simplified version — identical graphs = correct algebra.
